摩擦起电

探索电荷的奥秘，体验静电的魅力

物理原理

不同物质的原子核对核外电子的束缚力大小不同，当两个物体相互摩擦时产生电荷分离现象

电荷转移原理

  • 束缚力较小的物体容易失去电子带正电
  • 束缚力较大的物体容易得到电子带负电
  • 所带电的数量相同，电性相反
  • 带电体具有吸引轻小物体的性质

电荷强度指示器

正电荷 0
负电荷 0

电子转移过程

物质A

失去电子 → 正电荷

物质B

获得电子 → 负电荷

气球吸引易拉罐

实验器材

易拉罐 气球

实验步骤

  1. 1 向气球内充入适当气体
  2. 2 将充有适当气体的气球放在头发上快速摩擦
  3. 3 把易拉罐放在平整的桌面上，再将与头发摩擦了的气球靠近易拉罐

现象观察

会看到易拉罐随气球滚动，展示静电的吸引力

脱毛衣时的静电现象

小学生常见问题的科学解答

学生问题

"在黑暗处脱毛衣时，有时候会看到小火花，并听到噼啪的声音，有时候毛衣脱掉以后，内衬却吸附在身上，这是为什么？"

火花和声音的原因

  • 毛衣与内衬摩擦产生大量静电
  • 电荷积累到一定程度发生放电
  • 放电时产生火花和噼啪声
  • 黑暗中更容易看到微弱的火花

衣物吸附的原因

  • 不同材料带上相反电荷
  • 异种电荷相互吸引
  • 静电力使内衬紧贴身体
  • 干燥环境下静电更明显
